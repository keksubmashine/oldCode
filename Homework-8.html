Работас таблицами

Таблица создается тегом < table >. К этому тегу добавляется тег < border > чтобы задать границы.
Таблица в html создается порядно, чтобы создать ряд нужно задать тег < tr > в теле которого можно задать тег для создания заголовка или тег для создания ячейки.
Заголовок создается тегом < th >, ячейка создается тегом < td >.

Заголовок таблицы задается тегом < th >
